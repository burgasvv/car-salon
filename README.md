
# ___Car Rental API___

##  Версии
* JDK 17
* Maven 3.9.9
* SpringBoot 3.3.5

## Основной стек зависимостей
* Spring Security
* Spring Web
* Spring Data JPA
* Postgresql

## Описание
    API для регистрации пользователей и  аренды автомобилей

## Запуск приложения
* > Запустить приложение в среде разработки, compose.yaml или jar файл
* > Протестировать запросы в файле request-test.http
